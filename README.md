# VEC Project Website

Public landing page for the KTH × E.ON Virtual Energy Community research project.

**Live:** https://xinnnnnnnnnnn.github.io/vec-project-website/

## Stack
- Pure HTML + Tailwind CSS (CDN) + Lucide icons
- GitHub Pages deployment
- No build step

## Editing content
Edit `index.html` directly. Replace placeholder assets in `assets/` keeping the same filenames.
